# TSX ReactFull

    A React typescript Template functionalities.

### Steps to Work:

    1. git clone.

    2. run - npm i

    3. run - npm run dev - for running it on web browser.

    4. open web browser and paste link localhost:8080

    5. run - npm run build - for production bundle.

### Supported Libraries:

    1. Primereact
    2. Rearmed-css
    3. Jquery
    4. Bootstrap
    5. React-Bootstrap
    6. Datatables
